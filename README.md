# MathQuiz
This application was developed as part of my academic coursework, focusing on programming logic and interface development using MIT App Inventor.

This application presents a math quiz where two random numbers (between 1 and 15) and a random operation (addition, subtraction, or multiplication) are generated. The user must input the correct answer to progress and increase their score. If an incorrect answer is given, the app displays a "Game Over" screen with an option to restart the quiz.
